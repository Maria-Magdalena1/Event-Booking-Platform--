# Event Booking Platform

## Описание
Event Booking Platform е уеб приложение за управление на събития с microservice архитектура.  
Приложението позволява на потребителите да разглеждат, резервират и управляват събития.  
Основната функционалност е разделена между основното приложение и отделен microservice за анализ на данни.  

- **Main Application**: управлява потребителите, събитията и резервациите.  
- **Data Analysis Service (Microservice)**: анализира данни за събитията и позволява на администратора да извършва анализ чрез бутон в интерфейса.  

## Основни функционалности
- Регистрация и логин на потребители  
- Създаване, редакция и изтриване на събития  
- Резервация на места за събития  
- REST API за интеграция с външни приложения  
- Data Analysis Service за администратори

## Технологии
- Java 17  
- Spring Boot  
- Spring Data JPA  
- MySQL  
- Maven
